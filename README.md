# Skills Init Repo

Универсальный репозиторий для инициализации навыков любой агентной системы.

## Концепция

**НЕ хранилище всех навыков** — а навык самообучения.

Одной командой агентная система:
1. Читает каталог навыков
2. Анализирует свои потребности
3. Создаёт/устанавливает нужные навыки
4. Организует их структуру

## Использование

### Вариант 1: Прямая инструкция агенту
```
Прочти https://github.com/YOUR/skills-init-repo и инициализируй навыки для [платформа]
```

### Вариант 2: CLI ssi
```bash
ssi openclaw    # Инициализировать OpenClaw
ssi opencode    # Инициализировать OpenCode
ssi codex       # Инициализировать Codex
ssi auto        # Автоопределение
```

## Структура

```
skills-init-repo/
├── CATALOG.md           # Каталог навыков (машиночитаемый)
├── MASTER_INIT.md       # Мастер-навык самообучения
├── skills/              # Готовые навыки
│   ├── core/            # Базовые (git, coding, memory)
│   ├── communication/   # Коммуникации (email, messaging)
│   ├── research/        # Исследования (search, scrape)
│   └── automation/      # Автоматизация (mcp, web)
├── references/          # Референсы по платформам
│   ├── openclaw.md      # Специфика OpenClaw
│   ├── opencode.md      # Специфика OpenCode
│   └── generic.md       # Универсальные паттерны
└── scripts/
    └── ssi              # CLI для инициализации
```

## Расширение репозитория

1. Добавить навык в `skills/`
2. Зарегистрировать в `CATALOG.md`
3. При необходимости — создать ссылку на другой репозиторий

## Ссылки на внешние репозитории

В `CATALOG.md` можно указывать внешние источники:
```yaml
- name: coding
  source: https://github.com/example/coding-skills
  priority: high
```

Агент сам скачает и интегрирует.

## Версионирование

- Репозиторий версонируется через git tags
- Навыки могут требовать минимальную версию репозитория
- Backward compatibility поддерживается
