# Memory Management

## Структура

```
~/.openclaw/workspace/
├── MEMORY.md           # Долгосрочная память
├── memory/             # Дневники
│   └── YYYY-MM-DD.md   # Заметки за день
└── HEARTBEAT.md        # Периодические задачи
```

## MEMORY.md — Долгосрочная память

**Что хранить:**
- Ключевые решения
- Важные факты о пользователе
- Настроенные инструменты
- Изученные уроки
- Текущие проекты

**Формат:**
```markdown
## Раздел
- Краткая запись
- Важная информация

## Другой раздел
- ...
```

**Правила:**
- ≤200 строк (контекст ограничен)
- Периодически чистить устаревшее
- Обновлять при значимых событиях

## Daily Notes

```bash
# Создать заметку за сегодня
touch ~/.openclaw/workspace/memory/$(date +%Y-%m-%d).md
```

**Что писать:**
- Что сделано
- Что узнал
- Проблемы и решения
- Идеи

## Heartbeat

Периодически (каждые 30 мин) проверять:
- Входящие
- Календарь
- Обновления MEMORY.md

---

**Важно:** Память ограничена. Пиши только суть.
